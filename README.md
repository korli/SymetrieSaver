SymetrieSaver
=============

ScreenSaver for Haiku-OS based on Symetrie demo by rrrola (http://rrrola.wz.cz)


Build
-----

For gcc2 based systems:

	$ setgcc gcc2
	$ make

For gcc4 based systems:
	
	$ setgcc gcc4
	$ make

Installation
------------

Copy SymetrieSaver binary to:

	/boot/home/config/add-ons/Screen Savers
